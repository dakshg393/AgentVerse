
# 🤖 AgentVerce – AI-Powered Interview Assistant

**AgentVerce** is an AI-powered web application that simulates real-time technical interviews. It uses OpenAI's API to generate dynamic interview questions and responses, helping users practice and prepare for coding and behavioral interviews in an interactive way.

---

## 🚀 Features

* 💬 **Real-Time Chat Interface** – Simulates interviewer-candidate conversation
* 🧠 **GPT-Powered Q\&A** – Generates technical and behavioral questions dynamically
* 👤 **Role Selection** – Choose between HR, technical, or system design interviews
* 📋 **Feedback System** – Provides tips and model answers (optional feature)
* ⚡ **Fast, Responsive UI** – Built for smooth, mobile-friendly usage

---

## 🛠️ Tech Stack

* **Frontend**: React, Tailwind CSS
* **AI Backend**: OpenAI GPT API (chat completions)
* **State Management**: useState, useEffect (React Hooks)

---

## 🧪 How It Works

1. User selects interview type (e.g., HR, Technical)
2. The app sends prompts to the GPT API
3. GPT returns intelligent questions and feedback
4. User can interact as if in a live interview

## 🔧 Local Setup

```bash
git clone https://github.com/your-username/agentverce.git
cd agentverce
npm install
npm start
```

You’ll need an OpenAI API key:

1. Create a `.env` file in the root directory
2. Add your key:

   ```
   REACT_APP_OPENAI_API_KEY=your_api_key_here
   ```
